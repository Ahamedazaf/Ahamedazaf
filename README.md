> **Multi-Tenant SaaS Examination Platform** — Built & Deployed Solo

| Feature | Details |
|---|---|
| 🏫 **Live Clients** | SIP Abacus (Sri Lanka) + Phoenix International School (Puttalam) |
| 📊 **Exams Processed** | 78+ and growing |
| 🏗️ **Architecture** | Multi-tenant with fully isolated data per institute |
| 🔐 **Auth** | JWT — Role-based (Super Admin / Institute Admin / Student) |
| 🛡️ **Anti-Cheat** | Tab-switch detection, fullscreen monitoring, auto-submit on violation |
| 📈 **Analytics** | Recharts dashboards with Excel/PDF/PNG export |
| 🎨 **White-Label** | Per-institute branding, themes scoped at API middleware |
| 📥 **Bulk Import** | CSV/Excel student pipeline with automated email broadcast |

**Stack:** `React 18` · `Laravel 11` · `MySQL` · `JWT Auth` · `Tailwind CSS` · `Vite` · `Recharts`
